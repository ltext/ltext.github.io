Reduce Duplication

The Copy-paste command is an essential for anyone using a computer. However, every time you use it, you need to manually duplicate the data. With 位text, you can parameterize files themselves.

Built with Precision

With Lambda Calculus as its underlying engine, 位text features both generality and precision - it's Hindley-Milner type system gives us the best of both worlds; polymorphism and correctness. Function application is now our copy-and-paste command.

Open Source
